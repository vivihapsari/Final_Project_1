# _FinalProjectOne_API

Final Assignment Project Katalon Studio by Hactive8

Final Assignment ini menggunakan API dari website Bisatopup (www.bisatopup.co.id). API yang diguanakan adalah API Mockup, karena untuk yang production harus men-generate API key dan harus menghubungi admin Bisatop.

API yang digunakan dalam final Assignment ini meliputi :

1. Profile -> https://private-anon-fcee903571-bisatopup.apiary-mock.com/product/daftar/:id
2. Daftar Produk -> http://private-anon-fcee903571-bisatopup.apiary-mock.com/product
3. Detail Produk -> https://private-anon-fcee903571-bisatopup.apiary-mock.com/product/daftar/:id
4. Transaksi Pulsa, Token PLN, dll -> http://private-anon-fcee903571-bisatopup.apiary-mock.com/transaksi/beli
5. Cek Tagihan -> http://private-anon-fcee903571-bisatopup.apiary-mock.com/tagihan/cek
6. Bayar Tagihan -> https://private-anon-fcee903571-bisatopup.apiary-mock.com/transaksi/bayar
7. Riwayat Transaksi -> http://private-anon-fcee903571-bisatopup.apiary-mock.com/transaksi/history
